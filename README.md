## Task 04: Social Media Sentiment Analysis & Visualization

**Name:** Yash Patel  
**Internship:** CodeCraft  

---

### Objective
Analyze and visualize sentiment patterns in social media data
to understand public opinion and attitudes toward different
topics or brands.

---

### Description
This task analyzes social media sentiment using Twitter data.
Sentiment distribution is visualized to understand overall public
opinion, and brand-wise sentiment analysis is performed to compare
how users perceive different brands.

---

### Dataset
**Twitter Entity Sentiment Analysis Dataset**

**Dataset Source:**  
https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis

**Note:**  
The dataset files are not uploaded to this repository.  
Please download the dataset from the above link and place them
in the project folder before running the code.

---

### Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

---

### Methodology
- Loaded training and validation datasets  
- Assigned appropriate column names  
- Combined both datasets for analysis  
- Removed missing tweet entries  
- Analyzed overall sentiment distribution  
- Visualized brand-wise sentiment patterns for top brands
